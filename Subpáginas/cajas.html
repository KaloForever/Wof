Apuntes:

  • Un elemento de HTML en una pagina todo se considera como una caja rectangular
  • El contenido es el area donde se muestra el texto, imagen o cualquier otro contenido del elemento
  • El padding es el espacio entre el contenido y el borde de la caja
  • El margin es el espacio entre el borde de la caja y los elementos adyacentes, este espacio
    separa el elemento de otros elementos de la pagina
  • El borde es el borde

Box-sizing

Resulta que hay una propiedad llamada Box-sizing que define cómo se calcula el tamaño total de un elemento en CSS. Porq por defecto el navegador suma el padding y el border al tamaño que defina, lo que aveces suele "romper" los diseños. Esta propiedad evita que esto suceda un ejemplo sería:

Sin la propiedad box-sizing: border-box

width: 100px;
height: 100px;

En realidad mide 170px x 170px

Usando box-sizing: border-box:

width: 100px;
height: 100px;

Si mide 100px x 100px
