Emmet

Emmet es una herramienta integrada en VS Code que permite escribir código HTML y CSS de forma extremadamente rápida utilizando abreviaturas (atajos basados en sintaxis CSS). Ya viene preinstalada por defecto, por lo que no requiere descargas adicionales.

Crear una Lista con múltiples LI

Aquí se usa el > para indicar herencia y el signo * para multiplicar la cantidad de etiquetas LI, se usan las {} para escribir contenido

Ejemplo: ul>li{meow}*3

  • meow
  • meow
  • meow

Incluso puedo meter dentro una etiqueta a y usar el . para añadir clases

Ejemplo: ul>li.clase-ejemplo*3{wof}

  • wof
  • wof
  • wof

Luego si pongo el $, Emmet va a generar un número secuencial

Ejemplo: ul>li.clase-ejemplo*3{wof $}

  • wof 1
  • wof 2
  • wof 3

Medidas

Al definir las medidas de alguna propiedad, si escribo:

Un número simple como 10, se definirá en Píxeles (10px)

Si pongo 10p, se definirá como 10%

Con 10e se definirá como 10em

Todo esto se puede combinar para escribir propiedades y valores de forma muy rápida
